# 🧠 1. CNN for Image Classification

## 📌 Task
Build a Convolutional Neural Network (CNN) to classify images into correct categories using datasets like MNIST or CIFAR-10.

---

## ⚙️ Implementation Steps

### 1. Data Preprocessing
- Loaded dataset (MNIST / CIFAR-10)
- Normalized pixel values
- Reshaped input data for CNN

### 2. Model Architecture
- Convolutional Layers (≥ 2)
- ReLU Activation
- Max Pooling Layers
- Fully Connected Layer

### 3. Training
- Model trained on training dataset
- Loss and accuracy tracked

### 4. Evaluation
- Test Accuracy
- Training Accuracy
- Confusion Matrix

---

## 🔧 Experiments Performed
- Changed number of filters
- Modified kernel size
- Applied dropout
- Tuned learning rate

# 🧠 2. Sentiment Analysis using NLP

## 📌 Task
Build a model to classify text into positive, negative, or neutral sentiment.

---

## 📂 Dataset Options
- IMDB Reviews
- Twitter Sentiment Dataset
- Amazon Product Reviews

---

## ⚙️ Implementation Steps

### 1. Text Preprocessing
- Converted text to lowercase
- Removed punctuation and stopwords
- Tokenization performed
- Optional: Stemming / Lemmatization

---

### 2. Feature Extraction
- Bag of Words (BoW)
- TF-IDF
- (Optional) Word Embeddings (Word2Vec / GloVe)

---

### 3. Model Building
- Logistic Regression / Naive Bayes
- LSTM / Simple RNN (Deep Learning approach)

---

### 4. Evaluation
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

---

### 📊 Visualization
- Word Cloud for most frequent words

---

## 🛠 Tech Stack
- Python
- TensorFlow / PyTorch
- Scikit-learn
- NLP Libraries (NLTK / spaCy)
